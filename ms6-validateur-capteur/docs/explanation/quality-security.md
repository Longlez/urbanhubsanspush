# Qualité et sécurité

Le projet inclut des rapports de tests, une configuration Flake8 et un workflow GitHub Actions pour exécuter les analyses de qualité et de sécurité.

- Les tests unitaires vérifient la validation capteur et le flux trafic.
- Flake8 est utilisé pour la qualité du code.
- Un scanner Snyk peut analyser les dépendances Python.
